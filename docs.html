Prepare your files

  • Pattern Marker - marker, which is generated from an image, to create .patt file use this link . Make sure to set Pattern Ration to 0.90.
  • Barcode Marker - marker, which is generated in a range of numbers, for example from 0 to 63. You can create a barcode use this link . Make sure you have chosen right oprions:
  • Border size (% of marker width): 0.1
  • Barcode dimensions: 3х3
  • Markers have black borders.
  • .mtl .obj - model files. Both files should be prepared with their textures and correctly added to the project.

Setup your project

  1. Create new repository on GitHub.
  2. Upload your and required files to the repository. (Use "Upload files" button or drag and drop files from your device to the repository)
  3. Required file do not need to be modified. Place all of the folder and files in the same files hierarchy as they were downloaded.
  4. Files of your content should be acceptable from the index.html, which we will add later. If they are on the same level of file hierarchy you will need only to enter their name in the generator to define a path to them.

Working with the generator

  1. Upload your patterns files or choose barcodes which you will use in your project. This will generate a table which you can use to input other files of your content.
  2. You can choose what kind of content will be linked with each pattern. For one pattern you can select either model, video or image with one audio file.
  3. After you click submit button the text for the index.html will appear.

Publishing the App

  1. Go to your project repository and add a new file. Name it index.html and paste the given text inside it.
  2. From your repository go to Setting scroll down to GitHub Pages. In the Source section select branch main or master instead of None and click on Save. Для отримання додаткової інформації про розгортання на GitHub Pages відвідайте pages.github.com.
  3. After a few second your app will be published at https://[your_username].github.io/[repository_name]/.
